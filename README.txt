SECTION DECORATOR - Course Plan Software

How to run:
1. Unzip the folder.
2. Open the folder in VS Code.
3. Open index.html in your browser.
   OR install the VS Code extension "Live Server" and click "Go Live".

Main features:
- Intro page with "Create Plan" button.
- Course table with these columns:
  1. Course Name
  2. Day Name
  3. Schedule
  4. Section Name
  5. Faculty Name
- Animated colorful futuristic UI with a funny/childish vibe.
- Floating animated buttons.
- Conflict alarm sound if multiple courses overlap on the same day.
- Conflict panel shows exactly which courses are conflicting and the overlapping time.
- Complete button.
- PDF download button after completing the plan.

Schedule input examples:
- 8:30 AM to 11:10 AM
- 8:30 AM-11:10 AM
- 14:00 to 15:30

Day input examples:
- Sunday
- Sunday and Wednesday
- Monday, Thursday

Important note:
The app saves your course data in browser localStorage, so your data remains even after refreshing the page on the same browser.
